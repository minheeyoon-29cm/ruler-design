# Spinner

다운로드,업로드,요청 작업 처리와 같은 진행되고 있음을 시각적으로 표현합니다.

## 사용법


수정필요: component_usage_demo 컴포넌트는 수동 변환 필요 


Spinner는 다운로드, 업로드, 처리 등과 같은 시스템 작업의 진행을 시각적으로 표시하는 데 사용됩니다.

## Appearance


수정필요: multi_image 컴포넌트는 수동 변환 필요 


Spinner는 다운로드, 업로드, 처리 등과 같은 시스템 작업의 진행률을 시각적으로 표시하는 데 사용됩니다. 기본(default)외 onColor라는 외형을 제공하고 있습니다. 각 외형은 배경과의 충분한 대비를 고려하여 적절하게 사용하길 권장합니다.


## default


![Spinner Appearance : 기본](/images/components/spinner/spinner-preview-02-default.gif)


기본적으로 밝은 컬러의 배경, 요소에서 기본적으로 default를 사용할 수 있습니다. 검정색에 불투명도 라인이 원형으로 채워져 진행 상태를 표시합니다.


## onColor


![Spinner Appearance : onColor](/images/components/spinner/spinner-preview-03-oncolor.gif)


onColor는 페이지 배경 또는 요소에 강한 대비 색상이 있거나 복잡한 이미지가 있는 경우 사용됩니다.

## Anatomy

## Programmatic State


사용자 맥락에 의해 개발자가 코드로 조작하거나 설정하는 상태를 의미합니다.


| State | Value | Default | Description |
| --- | --- | --- | --- |
| isPending | boolean | false | true일 때 spinner가 재생됌 |
