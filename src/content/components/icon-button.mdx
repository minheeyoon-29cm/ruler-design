# Icon Button

사용자의 클릭 또는 터치를 통해 요구한 작업을 즉시 시작시키는 컨트롤입니다. Icon Button을 통해 사용자는 취소 및 선택, 페이지 이동과 같은 작업 수행에 조치를 취할 수 있습니다.

## 사용법


수정필요: component_usage_demo 컴포넌트는 수동 변환 필요 


Icon Button은 Text Label 대신 아이콘을 표시하게 됩니다. 버튼의 목적을 쉽게 이해할 수 있는 경우에만 Icon Button을 사용해주세요.

## 미리보기


## Hierarchy


![icon button Hierarchy](/images/components/icon-button/icon-button-01-hierarchy.jpg)


버튼의 Hierarchy는 서비스의 흐름 맥락에서 어떤 액션이 더 중요한지 강조하여 사용자의 작업 수행을 돕습니다.


a. primary : 사용자가 작업을 완료하는데 가장 중요한 액션에 사용됩니다. 강한 강조를 띄우고 있으므로 페이지 안에서 아껴서 사용해야 합니다. 
b. secondary : 약한 강조의 성격을 가지고 있어 primary보다 낮은 중요도의 사용자 액션에 사용될 수 있습니다. 
c. tertiary : 낮은 강조의 성격으로 주로 취소, 해제, 건너뛰기와 같이 사용자의 목표 달성과 무관한 액션에 사용될 수 있습니다. 


## isTransparent


![icon button Appearance](/images/components/icon-button/icon-button-02-appearance.jpg)


Icon Button은 isTransparent라는 영역은 존재하지만 배경색은 포함되지 않은 Appearance를 제공하고 있습니다. 


## Disabled


![Text button의 비활성화-Disabled 상태](/images/components/icon-button/icon-button-03-disabled.jpg)


비활성화 상태는 현재 어떤 이유로 편집, 변경이 불가하여 사용할 수 없는 상태를 나타냅니다. 이후 해당 항목을 다시 사용할 수 있다는 맥락을 사용자에게 전달할 수 있습니다.



## Size


![Text Button의 사이즈](/images/components/icon-button/icon-button-04-size.jpg)


Icon Button은 4가지 크기를 제공하며 Medium이 기본 값으로 자주 사용됩니다, 크기는 표현해야할 정보 구성에 따른 계층에 유의하여 사용해야 합니다

### WEB


기본적으로 버튼에는 hover와 pressed와 같이 사용자 행동으로 결정되는 속성들이 포함되어 있습니다. 이러한 속성은 사용자에게 즉각적인 피드백을 제공하며, 클릭 가능한 요소를 명확하게 기술하는 역할을 합니다

### Mobile


모바일에 경우 hovered 속성은 생략하고 pressed 속성으로 피드백을 제공합니다


![text button Keyboard Focus](/images/components/icon-button/icon-button-05-behavior06.jpg)


### Keyboard Focus


버튼은 키보드 Tab키로 탐색하여 Space 또는 Enter키를 통하여 버튼의 동작을 실행시킬 수 있습니다. 시각적으로는 파랑색 테두리를 추가하여 표현합니다.

## Programmatic State


사용자 맥락에 의해 개발자가 코드로 조작하거나 설정하는 상태를 의미합니다.


| State | Value | Default | Description |
| --- | --- | --- | --- |
| isDisabled | boolean | false | true일 때 비활성화 되어 선택할 수 없음 |


## Behavior State


주로 사용자 인터랙션과 관련이 있으며, 디자이너가 사용자 경험을 디자인할 때 중요한 역할을 합니다. 사용자는 이 상태를 직접적으로 경험할 수 있으며, 이를 통해 직관적이고 즉각적인 피드백 경험을 줄 수 있습니다.


| State | Value | Default | Description |
| --- | --- | --- | --- |
| hover | boolean | false | 마우스 오버 시 true로 전환 |
| pressed | boolean | false | 클릭 또는 탭시에 true로 전환 |
| focus | boolean | false | 키보드 포커싱 시 true로 전환 |
