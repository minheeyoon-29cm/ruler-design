# ProductCard V2

브랜드명, 상품명, 가격, 뱃지 등 핵심 정보를 시각적으로 표현하는 **모듈형 카드 컴포넌트**입니다.  
다양한 상품 리스트 및 콘텐츠 영역에서 활용됩니다.

---

## Purpose & Context

- ProductCard V1의 한계를 개선한 최신 버전
- 리스트 뷰, 추천 영역 등에서 정보 전달 최적화
- 브랜드 경험 및 플랫폼 간 UI 일관성 제공

---

## Component Overview

```tsx
<ProductCardV2
  brandName="Maison Kitsuné"
  productInfo={{
    name: 'Fox Head Patch Sweatshirt',
    price: 210000,
    saleRate: 15,
    badges: ['NEW'],
  }}
  metadata={{
    rating: 4.8,
    reviewCount: 201,
  }}
/>
```
