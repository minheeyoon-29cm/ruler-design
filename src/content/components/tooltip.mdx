# Tooltip

Tooltip은 UI 요소를 가르키며, 요소의 기능이나 정보를 설명하는 수단입니다. 
Tooltip은 일반적으로 짧은 시간 동안 표시되며 어떤 경우에는 사용자가 닫을 때까지 표시됩니다.

![Image](https://img.29cm.co.kr/next-edition/2024/08/18/dc179ee9de264a3e9f97aad5d03b43d8_20240818224956.jpg)

- 🅐 Trigger : Tooltip을 나타나게 하는 요소 또는 기준 
- 🅑 Pointer : Trigger를 가르키는 요소 
- 🅒 Dismissable : Tooltip을 제거할 수 있는 액션 
- 🅓 Contents : Label과 Dismissable을 포함하는 Contents

---

### Placement

![Image](https://img.29cm.co.kr/next-edition/2024/08/19/eae0057cf90c4d49a8b8a5857ab13a4e_20240819001020.jpg)

Tooltip은 Trigger기준으로 위치가 배치됩니다. 기본 배치 값은 `top` 입니다. 
Triggerd가 되는 Reference로 부터 기본 사이 간격 4px를 가지게 됩니다. 

![Image](https://img.29cm.co.kr/next-edition/2024/08/19/bcea256b5452453cb416bfd18867646c_20240819004133.jpg)

Tooltip은 Trigger가 되는 요소의 정보를 보조해줄 수 있어야 합니다. 
Trigger를 가장 잘 보조해줄 수 있는 위치에 Tooltip 배치해주세요.  

### Appearance

![Image](https://img.29cm.co.kr/next-edition/2024/08/18/ee000737804742b9a56ffc1f08d8d4a4_20240818234200.jpg)

Tooltip은 2가지 Appearance 스타일을 가집니다. 기본 스타일은 Gray입니다. 

![Image](https://img.29cm.co.kr/next-edition/2024/08/19/173a2d73b2fc402eaef496bc90b3c2ad_20240819013042.jpg)

특정 섹션에 검정색, 또는 색상 배경이 들어가는 경우 White의 Tooltip을 사용할 수 있습니다. 

### isDismissable

![Image](https://img.29cm.co.kr/next-edition/2024/08/18/0259d4b2c14143af8e7fbac17d85e8cc_20240818234200.jpg)

Tooltip은 Dismissable Action을 가질 수 있습니다. 

사용자가 Tooltip의 내용을 원하지 않을 때 언제든 Tooltip을 제거할 수 있도록 Dismissable을 통해 Tooltip을 제거할 수 있습니다.

### Contents Guide

![Image](https://img.29cm.co.kr/next-edition/2024/08/19/0db44a66245f4eff9c4ca623d424be3d_20240819051554.gif)

Tooltip은 기본적으로 사용자가 요구할 때 나타나야 합니다. 이 경우 사용자가 다른 액션을 취하기 전까지 Tooltip이 사라져선 안됩니다. 

사용자가 Scroll 또는 Dismissable Action을 실행할 때 Tooltip이 사라져야 합니다.

![Image](https://img.29cm.co.kr/next-edition/2024/08/19/c235e6977cc1481b80ab34ba0b1361c0_20240819141720.png)

Tooltip 에 사용자가 알아야 필수 정보를 넣지 마세요. 특정 Trigger로 인해 나타나는 Tooltip의 특성상 발견성이 낮으며 사용자가 다른 액션을 취하는 경우 사라집니다. Tooltip에는 사용자가 쉽게 인지 가능한 짧은 정보 텍스트만 포함해야 합니다.

대안이 필요한 경우 고정 위치에 안내 메시지를 포함하거나, 필수 여정에 포함하여 사용자에게 정보를 전달해주는 것이 이상적입니다. 

![Image](https://img.29cm.co.kr/next-edition/2024/08/19/b9cd2dcc507e41278daa0ab9f7e762e0_20240819182351.png)

툴팁에 링크나 다른 대화형 구성 요소를 넣지 마세요. 
사용자에게 유용한 정보는 일시적이더라도 고정적인 위치에서 사용자에게 명확하게 고지되는 편이 더 좋습니다.

![Image](https://img.29cm.co.kr/next-edition/2024/08/19/63be5340bf6748dd9dd6a78a2a5338aa_20240819034339.gif)

사용자에게 요구되지 않았지만 새로운 기능 혹은 전략적인 이유로 Tooltip을 사용하는 경우 사용자의 인지와 학습을 유도해야 합니다.

이 경우 무엇을 "인지했다" 또는 "학습했다"의 기준으로 삼을지 고려해 주세요. 
"인지", "학습"의 기준으로 삼을 수 있는 몇 가지 Metrics를 제안합니다.
