# Text Button

사용자의 클릭 또는 터치를 통해 요구한 작업을 즉시 시작시키는 컨트롤입니다. Text Button을 통해 사용자는 취소 및 선택, 페이지 이동과 같은 작업 수행에 조치를 취할 수 있습니다.

## 사용법


수정필요: component_usage_demo 컴포넌트는 수동 변환 필요 


Text Button은 화면안에서 부수적인 사용자의 작업 수행을 돕습니다. 화면의 전체적인 맥락과 밀접한 연관이 없거나 주목도가 낮은 작업에 Text Button을 사용하세요.

버튼은 사용자가 버튼을 선택할 때 어떤 동작이 일어날지 예측할 수 있어야 합니다. 행동을 장려하는 동사로 사용자를 이끌어주세요.


![textButton에 동사 명사를 명확히 표현하라는 지침](/images/components/text-button/usage/text-button-usage-00.png)


완료, 닫기, 취소, 확인과 같은 일반적인 작업을 제외하고는 동사와 명사를 명확히 표현해주세요.

## 미리보기


## Hierarchy


![text button Hierarchy](/images/components/text-button/preview/text-button-preview-00-hierarchy.jpg)


버튼의 Hierarchy는 서비스의 흐름 맥락에서 어떤 액션이 더 중요한지 강조하여 사용자의 작업 수행을 돕습니다.


a. primary : 사용자가 작업을 완료하는데 가장 중요한 액션에 사용됩니다. 강한 강조를 띄우고 있으므로 페이지 안에서 아껴서 사용해야 합니다. 
b. secondary : 약한 강조의 성격을 가지고 있어 primary보다 낮은 중요도의 사용자 액션에 사용될 수 있습니다. 
c. tertiary : 낮은 강조의 성격으로 주로 취소, 해제, 건너뛰기와 같이 사용자의 목표 달성과 무관한 액션에 사용될 수 있습니다.

## Icon


수정필요: multi_image 컴포넌트는 수동 변환 필요 


버튼과 함께 사용되는 아이콘은 각 위치마다 역할이 정해져 있습니다. prefix(접두사)와 postfix(접미사)의 의미를 주의하여 사용해주세요.


![Prefix Icon 사례 설명](/images/components/text-button/preview/text-button-preview-07-prefix.jpg)


### Prefix Icon


prefix icon은 레이블 왼쪽에 위치하며 레이블의 텍스트의 의미를 강조하거나 보조할 때 사용합니다.


![Postfix Icon 사례 설명](/images/components/text-button/preview/text-button-preview-08-action.jpg)


### Postfix Icon


postfix icon은 레이블 오른쪽에 위치하며 사용자의 행동과 상호 작용을 암시합니다.  페이지 이동, 바텀시트 등장이 있을 수 있습니다. 때문에 사용자가 기대하는 액션이 무엇인지 고려하여 사용하길 권장합니다.


## Behavior


![Touch targets](/images/components/text-button/preview/text-button-preview-09-begavior01.jpg)


### Touch targets


위와 같은 기준으로 터치 영역을 구축하면 손가락이 크거나 손 떨림, 장애가 있는 모든 사람들에게 도움이 될 수 있으며 한 손 또는 손가락의 일부로만 디바이스를 사용할 수 있어 터치 실수로 인한 사용자 이동을 최소화할 수 있습니다. 


수정필요: multi_image 컴포넌트는 수동 변환 필요 


### WEB


기본적으로 버튼에는 hover와 pressed와 같이 사용자 행동으로 결정되는 속성들이 포함되어 있습니다. 이러한 속성은 사용자에게 즉각적인 피드백을 제공하며, 클릭 가능한 요소를 명확하게 기술하는 역할을 합니다


수정필요: multi_image 컴포넌트는 수동 변환 필요 


### Mobile


모바일에 경우 hovered 속성은 생략하고 pressed 속성으로 피드백을 제공합니다


![text button Keyboard Focus](/images/components/text-button/preview/text-button-preview-09-begavior07.jpg)


### Keyboard Focus


버튼은 키보드 Tab키로 탐색하여 Space 또는 Enter키를 통하여 버튼의 동작을 실행시킬 수 있습니다. 시각적으로는 파랑색 테두리를 추가하여 표현합니다.

| --- | --- | --- | --- |
| isDisabled | boolean | false | true일 때 비활성화 되어 선택할 수 없음 |


## Behavior State


주로 사용자 인터랙션과 관련이 있으며, 디자이너가 사용자 경험을 디자인할 때 중요한 역할을 합니다. 사용자는 이 상태를 직접적으로 경험할 수 있으며, 이를 통해 직관적이고 즉각적인 피드백 경험을 줄 수 있습니다.


| State | Value | Default | Description |
| --- | --- | --- | --- |
| hover | boolean | false | 마우스 오버 시 true로 전환 |
| pressed | boolean | false | 클릭 또는 탭시에 true로 전환 |
| focus | boolean | false | 키보드 포커싱 시 true로 전환 |
