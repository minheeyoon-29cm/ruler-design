# Banner 

Banner는 모든 사용자에게 동일한 메시지와 정보의 맥락을 전달하는 역할을 합니다. 정보의 종류에 따라 높은 주목성을 위한 위치를 가질 수 있습니다.

![heroImage](https://img.29cm.co.kr/next-edition/2024/08/20/e8631c1cb9db48c0b74bb06c174c71f8_20240820022856.jpg)

### Anatomy

![Image](https://img.29cm.co.kr/next-edition/2024/08/20/8a42bee0bd6a4b6faa33fcdc703c281f_20240820030106.jpg)

-  Prefix Items : 좌측 영역을 뜻하며 `Icon`,`Image`,`Badge` 요소를 사용할 수 있습니다. 
-  Label : 사용자에게 전달해야할 텍스트 메시지를 작성할 수 있습니다, 
-  Postfix Items : 우측 영역을 뜻하며 이동과 닫그 동작을 유도하는 Affordance를 사용할 수 있습니다.  

![Image](https://img.29cm.co.kr/next-edition/2024/08/20/716f3d716edb4874be60cce0eb7d9550_20240820030106.jpg)

Prefix Items은 선택사항(Optional)으로 설계 의도에 따라 생략할 수 있습니다. 

### Tone 

![Image](https://img.29cm.co.kr/next-edition/2024/08/20/9a7976d9e7714a36bfa51405da113fb1_20240820030106.jpg)

`Tone`은 다양한 UI 메시지의 성격을 구분하여 사용자에게 더 명확하고 직관적인 피드백을 제공하기 위해 정의된 Props입니다. 
이 속성은 주로 UI의 알림, 상태, 경고 등을 시각적으로 구분하여 사용자의 주의를 효과적으로 유도하는 역할을 합니다. 

![Image](https://img.29cm.co.kr/next-edition/2024/08/20/6205633284064068b6efe6c80ae400e1_20240820030153.jpg)

각 Tone은 특정 상황에 맞는 색상과 메시지 스타일로 설정됩니다. 
이를 통해 사용자가 메시지를 보다 쉽게 이해하고, 상황에 적절하게 반응할 수 있도록 합니다.

- Default(기본) : 일반적인 정보 제공이나 안내 메시지를 표시할 때 사용됩니다. 
- Information(정보) : 중요한 정보나 사용자에게 주의를 기울이도록 알릴 때 사용됩니다.
- Positive (긍정적 피드백) : 긍정적인 피드백이나 성공적인 작업 완료를 알릴 때 사용됩니다.
- Negative (부정적 피드백) : 오류나 문제가 발생했을 때 사용자에게 이를 알릴 때 사용됩니다.

### hasPadding

![Image](https://img.29cm.co.kr/next-edition/2024/08/20/5cb84a57129f4ba68e67c95661b1e132_20240820030231.jpg)

`hasPadding`는 Banner의 시각 속성 중 하나로 UI에서 특정 메시지 또는 알림을 나타내기 위한 속성입니다. 

![Image](https://img.29cm.co.kr/next-edition/2024/08/20/921b6304cdfb424c8c6092b6cf60bd59_20240820022856.jpg)

정보의 성격에 따라 가시성이 높은 위치에서 사용자에게 적절한 안내가 필요한 경우 사용할 수 있으며, 주로 페이지 상단부에 위치됩니다.
